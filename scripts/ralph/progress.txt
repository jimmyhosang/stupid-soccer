## Codebase Patterns
- SvelteKit uses +page.svelte for pages, +page.server.ts for server-side load/actions
- Supabase client in src/lib/server/supabase.ts for server-side, src/lib/supabase.ts for browser
- Tailwind v4 with violet/emerald/slate color palette
- Phaser 3 game embedded as Svelte component
- All database tables MUST have RLS enabled
- Use Svelte 5 runes: $state, $derived, $effect
- API auth pattern: get sb-access-token cookie, verify with supabaseAdmin.auth.getUser()
